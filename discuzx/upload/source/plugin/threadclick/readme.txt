[帖内表态 1.0]
本插件以最纯粹的方式把日志或文章的表态功能搬到帖子里
绝大部分代码来 x2 自身的默认程序
一样的表态一样的展示

[安装方法]
按目录结构上传文件。
什么？你不懂什么叫按目录结构？那好吧。。。
你的论坛程序的目录下面有 source 等一堆目录, source 下又有 plugin 等一堆的目录，说到这里你再不小心看了一下这个插件目录
发现我这里也有 source ，下面也有 plugin ，对了你太聪明了 就是按相同的结构上传。

按目录结构上传文件，然后到后台 - 插件 里安装 并且在 插件列表开启。

[御载方法]
后台 - 插件 里面找到本插件点卸载
卸载会删除原来的所有本插件相关的表态数据，如果你确定是不使用这个插件否则不要卸载。
