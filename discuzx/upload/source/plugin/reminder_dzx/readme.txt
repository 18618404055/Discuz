[提醒者 1.1 beta]
在信息爆炸的时代，信息的不及时或不对等是可怕的。
你在身为论坛普通会员或者身为论坛站点的管理者时，会为论坛上不能及时得到关于你或者提及你消息而烦恼的话，
论坛次及时通提醒者插件可以为你解决此问题。

[安装方法]
按目录结构上传文件。
什么？你不懂什么叫按目录结构？那好吧。。。
你的论坛程序的目录下面有 source 等一堆目录, source 下又有 plugin 等一堆的目录，说到这里你再不小心看了一下这个插件目录
发现我这里也有 source ，下面也有 plugin ，对了你太聪明了 就是按相同的结构上传。

按目录结构上传文件，然后到后台 - 插件 里安装 并且在 插件列表开启。

[升级方法]
按目录结构上传覆盖，然后在后台点升级。

[御载方法]
后台 - 插件 里面找到本插件点卸载
卸载会删除原来的所有本插件相关的数据，如果你确定是不使用这个插件否则不要卸载。
