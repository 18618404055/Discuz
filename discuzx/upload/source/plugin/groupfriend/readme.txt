友情群组插件

1、插件的作用：
交换群主创建的群组
群组友情链接模式呈现


2、设计思路：
当一个群组访问他人创建的群组时，可以对别人的群组加为友情群组，那样在群组右边栏就会出现一个友情链接形式的群组标志，其他的群用户可以通过这个标志访问链接中的群组。

群组交换是单方面，也就是说，当自己主动添加了对方的群组，只在自己的群中存在链接。系统会发出一个短消息通知对方群主，建议他也添加您的群。形成交换的目的

3、安装方法：
1、将upload文件包中的目录上传到网站的根目录。
2、进入站点后台 -> 插件 -> 安装新插件 -> 安装
3、安装完毕之后启用即可。
For Discuz! X1.5 (正式版和RC版本均可)
