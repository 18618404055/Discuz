友情群組插件

1、插件的作用：
交換群主創建的群組
群組友情鏈接模式呈現


2、設計思路：
當一個群組訪問他人創建的群組時，可以對別人的群組加為友情群組，那樣在群組右邊欄就會出現一個友情鏈接形式的群組標誌，其他的群用戶可以通過這個標誌訪問鏈接中的群組。

群組交換是單方面，也就是說，當自己主動添加了對方的群組，只在自己的群中存在鏈接。系統會發出一個短消息通知對方群主，建議他也添加您的群。形成交換的目的

3、安裝方法：
1、將upload文件包中的目錄上傳到網站的根目錄。
2、進入站點後台 -> 插件 -> 安裝新插件 -> 安裝
3、安裝完畢之後啟用即可。
For Discuz! X1.5 (正式版和RC版本均可)
